HTML

HTML, cuyas siglas significan HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje principal utilizado para crear la estructura de las páginas web. A diferencia de los lenguajes de programación, HTML no contiene lógica ni algoritmos; su función principal es organizar y dar formato al contenido que será mostrado en un navegador.

Estructura de Lenguaje

HTML funciona mediante el uso de "etiquetas", que son instrucciones que indican al navegador cómo debe organizar el contenido. Estas etiquetas envuelven el texto, imágenes, enlaces y otros elementos, definiendo su propósito y posición en la página. Cada etiqueta tiene una función específica, como mostrar un encabezado, un párrafo o un enlace.

Comunicacion del Navegador

Cuando un navegador web abre una página, lo primero que hace es leer el archivo HTML. El navegador interpreta las etiquetas y presenta el contenido de forma visual según el significado de cada una. Por ejemplo, si una etiqueta indica que cierto texto es un título, el navegador lo mostrará en un tamaño y estilo diferente al texto normal.

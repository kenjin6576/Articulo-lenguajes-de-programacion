Java

Java es un lenguaje de programación de propósito general, orientado a objetos, que se utiliza para desarrollar aplicaciones en una gran variedad de entornos: desde programas de escritorio hasta aplicaciones móviles, sistemas empresariales y servidores web. Fue creado por Sun Microsystems en los años 90 y actualmente es mantenido por Oracle.

Filosofia de Java

Una de las características más importantes de Java es su capacidad de ser portátil. Esto significa que un mismo programa escrito en Java puede ejecutarse en diferentes sistemas operativos (como Windows, macOS o Linux) sin necesidad de ser reescrito. Esto es posible gracias a la JVM (Java Virtual Machine), que actúa como un intermediario entre el programa y el sistema operativo.

¿Que es la JVM?

La JVM es una máquina virtual que interpreta y ejecuta el código Java. Cuando se compila un programa en Java, no se traduce directamente a lenguaje de máquina, sino a un formato intermedio llamado bytecode. Este bytecode puede ser interpretado por cualquier JVM, lo que permite que el mismo programa funcione en distintos dispositivos sin cambios en el código fuente.
